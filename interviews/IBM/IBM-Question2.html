Given the problem below Write a function to solve it // Input: 100, Expected output: 112 // Input: 12, Expected output: 15 // Input: 123, Expected output: 168 function getExpressionSums(num: string): number { const MOD = 1000000007; // 10^9 + 7 const n = num.length; // For n digits, we have (n-1) positions between digits where we can insert '+' // Total number of combinations: 2^(n-1) let totalSum = 0; // Generate all possible combinations using bit manipulation // Each bit represents whether to insert '+' at that position const numCombinations = 1 << (n - 1); // 2^(n-1) for (let mask = 0; mask < numCombinations; mask++) { let currentSum = 0; let currentNumber = 0; // Build number modulo MOD as we go // Build the expression based on the mask for (let i = 0; i < n; i++) { const digit = parseInt(num[i]); // Build number digit by digit: currentNumber = currentNumber * 10 + digit currentNumber = (currentNumber * 10 + digit) % MOD; // Check if we should insert '+' after this digit // (if this is not the last digit and the corresponding bit is set) if (i < n - 1 && (mask & (1 << i)) !== 0) { // Insert '+' here currentSum = (currentSum + currentNumber) % MOD; currentNumber = 0; // Reset for next number } } // Add the last number currentSum = (currentSum + currentNumber) % MOD; totalSum = (totalSum + currentSum) % MOD; } return totalSum; } Problem:

Given a string num consisting of digits ('0' to '9'), find the sum of the values of all possible expressions formed by inserting '+' characters between digits.

 

Rules:

  1. You can insert '+' characters between any two digits.
  2. No adjacent '+' characters are allowed.
  3. The expressions are evaluated as regular mathematical expressions.

Since the answer can be large, return the value modulo (109 + 7).

 

Example

num = "123"

 

All possible valid expressions:

  • "1+23" = 24
  • "12+3" = 15
  • "1+2+3" = 6
  • "123" = 123

Sum: 24 + 15 + 6 + 123 = 168

Answer: 168 modulo (109 + 7) = 168

 

Function Description

Complete the function getExpressionSums in the editor with the following parameter:

    string num: the string of numbers

 

Returns

    int: the sum of all possible expression values modulo (109 + 7)

 

Constraints

  • 1 ≤ length of num ≤ 17
  • num consists of characters from '0' - '9' (both inclusive) only.
